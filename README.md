# AIK — Digital Product Studio

A premium, interactive portfolio website for AIK Studio.

## Structure
- `index.html`: Main entry point.
- `css/style.css`: Comprehensive design system and styles.
- `js/data.js`: Structured content data.
- `js/script.js`: Main logic for rendering and interactions.

## Features
- Dynamic content generation from `data.js`.
- Custom interactive cursor.
- Scroll-triggered reveal animations.
- Premium dark-themed aesthetic.
